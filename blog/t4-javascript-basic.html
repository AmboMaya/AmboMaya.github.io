How does JavaScript compare to HTML and CSS?

Javascript (JS) is an object oriented programming language wherease HTML is a markup language whose job is to display it to the eyes of the user. With JS you can add functionality or behaviour to the web application. Not only JS is used to make a website to visuallly look attractive and interactive, it can also be used on the server side i.e how your website can be available on the webservice and how people use your site. A more technical feature JS can do is that it can change the type of variable or add new properties or methods to an object while the programming is running.

Explain the difference between accessing data from arrays and objects

Objects and arrays data are represented differently. An object is surounded by curly braces { } and an array surrounded by square brackets [ ]. Data arrays can be accessed by [ ] with its index specified inside the brackets or the value. On the other hand, to access an object's property we need to use a period sign (.) followed by variable name and name of the property. See example below:

var myFavouriteNumber = [32, 25, 27, 40]. To access the data of students array, student[0] which will return number 32.
var student = { name: "maya", age: 32, city:"auckland"} to access property age of the object student: student.age will return 32.
