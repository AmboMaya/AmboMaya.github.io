Grid Based Design

It's a conventional pattern for designing a webpage i.e the design flow starts from the top left of the page, down to the left of the page. The layouts are broken down into grids.

Hype About Responsive Webpages

With the revolution of internet, it's a lot easier to browse the internet from different types of devices, such as mobile phones, tablets, laptop, etc. These devices have different screen sizes. A responsive webpage provides a good user experience as it's easier for the user to open web app on whatever device they have. The layout of responsive webpage is adjustable according to device screen size on which the user access page from.

Semantic structure

A webpage uses semantic structure where HTML elements are used for the correct purpose. For example, not to use href on button tag. A semantically structured webpage will have less or none redundant tags. In the Udacity HTML tutorial, they mentioned that a wepage with semantic codes load a lot faster, and it is also easier to read and for other people to understand.
