HTML, CSS and DOM Part 1

Explanation or Analogy

HTML is markup langauage whose job is to display your webpage to the users through a web browser. It displays the look of your site based on the structure using tree-like structuring method. The structure represents by tags also knowns as syntacs.

CSS is responsible for styling your website making attractive for a better user experience for your audience.

DOM is the browser interpertation of you HTML documents. Like HTML, DOM also respresents the document in the browser with tree-like structure.

Boxifying Design

HTML elements layout are wrapped in rectangular boxes. To get a better understanding on a webpage layout you can use boxifying method where you can segment each every element that visually capture into single boxes.
