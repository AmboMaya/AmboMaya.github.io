Symbols use cases in javascript
  • brackets [ ] : are used for accessing the properties of an Object (or the elements of an Array).
  • braces { } : are used during the declaration of Object Literals, or to enclose blocks of code (function definitions, conditional blocks, and loops).
  • single quotes '' and double quotes " " :

    • Both single and double quote are used for storing and manipulating text. If you want to use quote mark within your string, you can use a single quote and use double quote for the word inside your string. For example "Go to the 'try-me' section".
